# StudyMatch - 学習記録・目標管理アプリ

StudyMatchは、日々の学習内容と学習時間を記録し、目標までの進捗を見える化するWebアプリです。

## 概要

学習内容と学習時間を入力すると、学習履歴として一覧表示されます。  
また、今日の合計学習時間と目標達成率を表示し、学習の継続をサポートします。

## 制作背景

TypeScriptの学習と、就職活動で提示できる制作物を増やすことを目的に制作しました。  
単に記録するだけでなく、学習時間を可視化することで、ユーザーが継続しやすい体験を意識しています。

## 使用技術

- HTML
- CSS
- TypeScript
- localStorage

## 主な機能

- 学習内容の追加
- 学習時間の記録
- 学習履歴の表示
- 学習記録の削除
- 今日の合計学習時間表示
- 目標達成率の表示
- localStorageによるデータ保存

## 工夫した点

- TypeScriptを使用し、学習記録のデータ型を定義しました。
- localStorageを使用し、ブラウザを更新しても履歴が残るようにしました。
- 目標達成率を進捗バーで表示し、学習状況を直感的に把握できるようにしました。
- シンプルで見やすいUIを意識してデザインしました。

## 今後の改善点

- 日付ごとの学習履歴表示
- カテゴリ別の学習時間管理
- 週間・月間の学習時間グラフ
- 目標時間をユーザーが自由に設定できる機能
- レスポンシブデザインの改善
